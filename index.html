Goals:

Complete all the tasks below:

  • Download all the resources (img, sound, font,...)

  • Set up a Pygame File (screen,caption, loop,...)

  • Set up Game Objects (Player, Enemy, Background, ...)

  • Give Logic to the game, then add jump_sound whenever the player jump.

  • Think about how to make a score calculator ? (Using Time Count)

  • Create a function named display_score to display the score. This function must:
    1. Using a method named pygame.time.get_ticks(). This method will return a value that's the time from when the game start.
    2. Create a variable, its value is alike with a text in Pygame (Hint: using font.render)
    3. Display the score by using screen.blit

  • If the player collide with enemy, show the Yellow screen.

  • Show your score when you lose the game
